Descargar una web para visualización local con wget

El comando es 
$ wget -E -H -k -K -p  http://www.google.es
Con -p hacemos que se descarguen los enlaces requeridos (css, javascript), con -k convertimos los enlaces dentro del html para visualización local (-K copia de seguriad) y con -E aseguramos que los ficheros descargados tengan extensión html para la visualización en local.
