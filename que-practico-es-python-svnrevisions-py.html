¡Que práctico es python!: svnrevisions.py

Pongo a continuación un script rápido que hice hace ya un tiempo. En ese momento estaba realizando un backport de código y de forma sucia realicé varios 
svn up -rTAL
De esa forma y con unos pequeños cambios obtuve el cambio que necesitaba. Pero después de un rato no recordaba que fichero cambié exactamente y a qué revisiones. Un poquito de terminal y listo. Se que es sucio, que podría usar las librerías de python para subversion, etc. Pero se hizo de forma rápida y funciona. El funcionamiento se puede ver en el código fuente, pero pongo una ayudita: 
$ python svnrevisions.py
$ python svnrevisions.py 123
$ python svnrevisions.py ALL
Aquí dejo el código: svnrevisions
